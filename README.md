## 🎮 游戏玩法

### 基本规则
1. 黑棋先行，双方轮流在棋盘交叉点上落子
2. 先在横、竖、斜方向连成五子者获胜
3. 棋盘填满未分胜负则为平局

### 操作指南
- **鼠标点击** - 在棋盘交叉点落子
- **触摸点击** - 在移动设备上触摸落子
- **齿轮图标** - 打开/关闭设置面板
- **语言切换** - 左上角"文/A"按钮切换中英文
- **悔棋按钮** - AI有明显优势时显示悔棋提示

## 🛠️ 技术实现

### 技术栈
- **HTML5** - 语义化结构
- **CSS3** - Flexbox布局、CSS动画、渐变效果
- **原生JavaScript (ES6+)** - 面向对象编程

### 核心模块
游戏采用模块化设计，包含以下核心类：

1. **GameState** - 游戏状态管理
   - 棋盘状态存储
   - 当前玩家管理
   - 游戏结束判断
   - 历史记录保存

2. **BoardManager** - 棋盘逻辑控制
   - 落子有效性验证
   - 获胜条件检查
   - 棋盘状态查询
   - 可行位置计算

3. **AIEngine** - AI智能算法
   - 启发式搜索
   - 位置评估函数
   - 棋型模式识别
   - 最佳落子决策

4. **UIManager** - 用户界面管理
   - 棋盘渲染
   - 事件处理
   - 状态更新
   - 动画效果

5. **GameController** - 游戏流程控制
   - 模块协调
   - 游戏初始化
   - 事件绑定
   - 流程控制

6. **I18nManager** - 国际化支持
   - 多语言翻译
   - 界面本地化
   - 语言切换

### AI算法特点
- **启发式评估** - 基于棋盘位置评分
- **模式识别** - 识别活三、冲四等关键棋型
- **防御优先** - 优先阻止玩家连珠
- **策略思考** - 考虑多步后的局面

### 响应式设计策略
1. **自适应布局** - 根据屏幕尺寸调整棋盘大小
2. **触摸优化** - 移动端触摸事件处理
3. **横竖屏适配** - 不同方向不同布局
4. **性能优化** - 减少重绘，使用CSS硬件加速
